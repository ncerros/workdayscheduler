# workdayscheduler
##
This daily planner provides an easy way to add and organize a 
busy daily schedule. With this planner by clicking the button,the user can save an events during each hour. 

### 
Libraries:
- Moment.js
- Jquery
- Bootstrap
- Fontawsone
- fontgoogle
